Detailed Description

Class to perform a grid search optimization for the offset and slope parameters of one harmonic drive scaling function.

The GridSearch class performs a grid search optimization for the offset and slope parameters of one harmonic drive scaling function. The grid search iterates over the specified ranges with the specified granularities. For every iteration, the search computes a harmonics calculation for these parameters and evaluates the specified criteria.
