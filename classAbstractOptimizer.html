◆ computeMagnetEllBounds()

void AbstractOptimizer::computeMagnetEllBounds ( )
protected

Compute and set the ell bounds for the magnet.

This function computes the ell bounds for the magnet by running a mesh calculation and sets them in the cct_ell_bounds_ variable. The ell bounds describe the part along the harmonic calculation's relative ell axis where the magnet actually is.
